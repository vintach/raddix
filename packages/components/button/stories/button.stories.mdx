## Features

- Full support for browser features.
- Full keyboard navigation.

## Installation

Install the primitive from your command line.

```sh
yarn add @raddix/button
```

## Anatomy

Import all or parts of <code>@raddix/button</code>

```jsx
import useButton from '@raddix/button';

const Button = ({ children, ...props }) => {
  const { elementProps } = useButton.Root(props);

  return <button {...elementProps}>{children}</button>;
};
```
